--- layout: post title: Graphics Project 3 tags: - 3d modeling - computer science - graphics - opengl - school status: publish type: post published: true meta: _edit_last: '1' --- Not long ago we got our third graphics project assigned. In the first we wrote our own raytracer, and in the second we got familiar with OpenGL and implemented a trackball interface that would load a model and allow the user to rotate it around to see it from different angles. This third project deals with texturing, however. The first part was to generate a torus and texture it with a tile-able pattern (we were given a few wood textures and one brick). Same trackball interface applies. This one seems to be the inside of a building near a window. [caption id="attachment_131" align="alignnone" width="300" caption="Bunny of mirrors"][/caption] Lastly, given a 3D texture (marble), we were to "carve" a model out of that material.
